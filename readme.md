## RxJS Tryout

- Frontend Library - RxJS
- Build Tool - Webpack + TsLoader
- Build Source Lang - TypeScript
- Hosted at Back4App

To Test This

- Install Package Dependencies
```sh
$ cd tm-rxjs-yt
$ yarn install

# or, with npm 
$ npm install
```

- Run App in PROD Mode
```sh
$ cd tm-rxjs-yt
$ yarn run start-app
```

- Run App in DEV Mode
```sh
$ cd tm-rxjs-yt
$ yarn run webpack-dev

# Remember to run 
$ yarn run webpack-build
#after changing the source
```
